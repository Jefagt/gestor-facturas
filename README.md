# Gestor de Facturas

## 📌 Descripción
Sistema básico para administrar clientes, productos y facturas.  
Incluye funciones de CRUD (crear, leer, actualizar, eliminar) y generación de reportes.  
Pensado como ejemplo de aplicación empresarial.

## 🛠️ Tecnologías
- C# (.NET)
- MySQL

## 🚀 Funcionalidades
- Registro de clientes y productos
- Emisión de facturas
